标题： 2017.02.09 socket群聊和私聊
课程内容： 


1、私聊的步骤
    a、客户端发消息给服务器，用对象保存数据
        data
	{
	  form:
	  to:
	  text:
	}
     b、服务器解析收到的消息
        data.to 是要发送给他 这是一个昵称
	根据data.to（昵称），找到该客户端（浏览器）的socket.id
        socket用to发送指定的id；
     c、服务器发送消息给找到的socket.id
        {
	  from:
	  to:
	  msg:
	  time:
	}

     d、获取在线列表
	流程：客户端请求在线列表
	      服务端把在线列表的名称返回给客服端


2、群聊
   直接发送给所有在线用户用io.emit
